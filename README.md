# My First Coding App

A beginner-friendly Flutter application that introduces basic programming concepts. This project serves as a starting point for learning how to build user interfaces, manage state, and handle button interactions in Flutter.

## Features
- **Interactive Flashcards:** Cycles through fundamental programming concepts like Variables, Functions, Strings, and Loops.
- **State Management:** Demonstrates the use of `setState` in a `StatefulWidget` to update the screen interactively.
- **Responsive Layout:** Centers content and uses flexible padding to look good on mobile devices, web, and desktop.

## Tech Stack
- **Framework:** Flutter
- **Language:** Dart
- **Design System:** Material Design 3

## Setup and Run
To run this project locally:

1. Make sure you have the [Flutter SDK](https://docs.flutter.dev/get-started/install) installed.
2. Clone this repository and open the project directory in your terminal.
3. Run `flutter pub get` to fetch any dependencies.
4. Run `flutter run` to launch the app on your connected device or emulator.
